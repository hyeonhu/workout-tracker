# 운동 트래커

근비대 목표의 4분할 운동 트래커입니다. React, Vite, Tailwind CSS, Firebase Firestore, Anonymous Auth, PWA, GitHub Pages 배포를 기준으로 구성되어 있습니다.

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run check
npm run build
```

GitHub Pages 배포용 `base`는 `vite.config.js`에 `/workout-tracker/`로 설정되어 있습니다.

## Firebase

Firestore Rules에는 `firestore.rules` 내용을 적용하세요. 복구 코드를 이용한 데이터 재연결을 위해 `userAccess` 컬렉션 권한도 포함되어 있습니다.

필요한 Firebase 기능:

- Authentication: Anonymous provider 활성화
- Firestore Database 생성

## PWA

iPhone Safari에서 배포 URL을 연 뒤 공유 버튼에서 “홈 화면에 추가”를 선택하면 앱처럼 사용할 수 있습니다.

## 중량 기준

- 바벨: 한쪽 원판 무게 기준
- 덤벨: 덤벨 개당 무게 기준
- 머신/케이블: 기구에 표시된 무게 또는 한 칸 기준

설정 화면에서 종목별 증량폭을 바꿀 수 있습니다.

## 세션 아코디언

오늘 운동 화면은 A1, B1, A2, B2 세션을 접고 펼칠 수 있는 아코디언으로 보여줍니다. 기본적으로 오늘 배정된 세션만 펼쳐지고, 접힌 상태에서는 요일, 종목 수, 총 세트 수, 무릎 민감 종목 여부를 빠르게 볼 수 있습니다.

기록 화면에는 `지난값`, `전부 +1`, `첫세트`, `하한` 빠른 입력 버튼과 완료 전 요약이 있어 운동 중 입력 탭 수를 줄입니다. 설정 화면은 기본적으로 오늘 세션만 보여주고, 버튼으로 전체 세션 중량을 볼 수 있습니다.

## 히스토리 아코디언

히스토리는 날짜별로 묶이고 기본 접힘 상태입니다. 날짜를 열면 완료한 세션, 종목별 세트 x 반복 x 중량, 메모, 세션 총 볼륨을 확인할 수 있습니다.

## 차트 기준

히스토리의 `대시보드`를 펼치면 주요 차트를 볼 수 있습니다.

- 주요 운동 진행: 벤치프레스, 시티드 덤벨 숄더프레스, 랫풀다운, 레그프레스, 루마니안 데드리프트, 힙쓰러스트를 봅니다.
  - 벤치프레스, 시티드 덤벨 숄더프레스, 루마니안 데드리프트는 best-set e1RM 기준입니다.
  - 랫풀다운, 레그프레스, 힙쓰러스트, 레그컬 같은 운동은 작업중량, 총 반복수, 세트 수를 봅니다.
- 체중 추세: 체중을 직접 기록하고 최근 10주 주간 평균을 봅니다.
- 주간 수행: 주당 완료 세션 수를 작게 보여줍니다.
- 고급 분석: 근육군별 주간 kg 볼륨 추세를 선택적으로 확인합니다.

볼륨은 기본적으로 `중량 x 총 반복수 x 근육군 기여도`로 계산합니다. 플랭크처럼 체중/시간 운동은 코어 수행량이 보이도록 중량 1로 계산합니다.

## 공유 중량과 세션별 정체

같은 운동 변형은 working weight와 증량폭을 공유합니다. 예를 들어 A1 레그프레스와 A2 레그프레스는 같은 중량을 씁니다. 대신 정체 카운트, 지난 반복수, 현재 세트 수는 세션 인스턴스별로 따로 유지합니다.

공유 중량은 앵커 세션에서만 올라갑니다.

- 레그프레스: A1 앵커, A2는 같은 중량 사용
- 레그컬: B1 앵커, B2는 같은 중량 사용, 햄스트링 민감 종목
- 랫풀다운: A1 앵커, B2는 같은 중량 사용
- 뉴트럴그립 랫풀다운은 별도 운동
- 시티드 케이블 로우와 체스트 서포티드 로우는 별도 운동

무릎 민감 운동인 레그프레스와 레그 익스텐션은 상한 반복을 달성해도 바로 증량하지 않고, 다음 앵커 세션에서 통증/부종/불안정이 없다고 확인해야 증량합니다.

햄스트링 민감 운동인 레그컬도 같은 느린 진행 철학을 씁니다. 다음날 날카로운 뒤허벅지 통증, 비정상적인 국소 근육통, 회복 지연이 없다고 확인해야 정상 증량합니다.

## 정체 추천

앱은 최근 4주를 기준으로 로컬 정체와 전반 정체를 구분합니다.

- 로컬 정체: 수행률이 충분하고 체중이 유지/소폭 증가하는데 특정 앵커 지표가 거의 변하지 않으면 세트 추가를 제안합니다.
- 무릎/햄스트링 민감 종목: 세트 추가보다 같은 중량으로 1주 더 관찰하거나 증량 보류를 먼저 제안합니다.
- 전반 정체: 여러 주요 운동이 같이 정체되거나 체중이 떨어지는 흐름이면 디로드, 식단, 회복 점검을 제안합니다.
